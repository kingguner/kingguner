github在国内时不时登录不上去的解决方法

发表于2022-02-02|更新于2021-08-22|技术分享
|阅读量:

前言

由于github的网站服务器在美国，在国内经常抽风，有时一直登录不上去，在网上找了很多方法，网上各路大神给的方法就是更改hosts,目前觉得下面的方法还是比较靠谱的。

更改hosts文件，地址如下：

C:\Windows\system32\drivers\ect

通过两个网站查到自己的IP, 网站如下：

http://github.com.ipaddress.com/ 
http://github.global.ssl.fastly.net.ipaddress.com/

将代码放到hosts末尾，保存、退出，格式如下：

Github Start
140.82.113.4 github.com
199.232.69.194 github.global.ssl.fastly.net
Github End

win+R打开cmd,输入如下代码刷新缓存

ipconfig /flushdns (注意：ipconfig后面有一空格)

结束
