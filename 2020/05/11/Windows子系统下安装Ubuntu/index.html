Windows子系统下安装Ubuntu

发表于2020-05-11|更新于2020-05-25|技术分享
|阅读量:

Windows 10系统下是可以通过Microsoft store安装Ubuntu的，步骤如下：
首先说明一下我的系统是Windows10专业版的，至于家庭版我没试过，好吧，startup

一. 启动Windows功能

右击Windows开始键→搜索→搜索“功能”→启动或关闭Windows功能→在适用于Linux的Windows子系统前面打勾。

二. 在微软应用商店Microsoft store中安装Ubuntu

左击Windows开始键→启动Microsoft store→搜索Ubuntu并下载安装，是不是很简单啊

结束
